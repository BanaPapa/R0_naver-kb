개인정보처리방침

시행일: 2026년 7월 8일

본 방침은 Estate-OS 웹 서비스(estate-os.xyz)와 크롬 확장 프로그램 "Estate-OS 매물시세 연결기"가 이용자의 정보를 어떻게 처리하는지 설명합니다.

핵심 요약. 확장 프로그램이 사용하는 네이버 로그인 쿠키·인증 토큰은 이용자의 브라우저 안에서만 사용되며 개발자 서버로 전송·저장되지 않습니다. 부동산 검색 결과 자체는 서버에 저장하지 않습니다. 어떠한 개인정보도 판매하지 않습니다.

1. 브라우저 확장 프로그램이 처리하는 정보

네이버가 데이터센터 IP를 차단하므로, 확장 프로그램은 이용자의 브라우저(주거 IP)에서 부동산 매물·시세·리뷰 데이터를 대신 조회해 웹앱에 전달합니다. 이 과정에서:

  • 네이버 세션 쿠키 — 이용자가 브라우저에 로그인해 둔 네이버 쿠키를 네이버 부동산 API 요청에 함께 실어 자동 차단(rate limit)을 완화합니다. 이 쿠키는 브라우저 안에서만 사용되며, 개발자 서버로 전송되거나 저장되지 않습니다.
  • 인증(Authorization) 토큰 — 네이버 new.land 요청에서 캡처해 브라우저의 임시 저장소 (chrome.storage.session)에 보관하며, 브라우저 세션이 끝나면 삭제됩니다. 외부로 전송하지 않습니다.
  • 요청 대상 — 모든 조회 요청은 이용자의 브라우저에서 네이버(*.naver.com)와 호갱노노(hogangnono.com)로 직접 전송됩니다. 개발자 서버를 경유하지 않습니다.

확장 프로그램은 위 정보를 수집·저장·판매하거나 제3자에게 제공하지 않습니다.

회원제 서비스 운영을 위해 다음 정보를 수집·저장합니다(저장소: Supabase).

항목내용목적
계정 정보 이메일, 이름, 회사/소속, 직급, 전화번호(선택) 회원가입, 관리자 승인, 본인 확인, 서비스 제공
검색 활동 로그 검색 시각, 이용한 탭, 검색 조건 요약(지역·유형 등), 결과 건수

3. 보관 기간 및 파기

  • 검색 활동 로그는 수집일로부터 6개월 경과 시 자동 삭제됩니다.
  • 계정 정보는 회원 탈퇴 또는 삭제 요청 시 지체 없이 파기합니다.

4. 제3자 제공 및 처리 위탁

이용자의 개인정보를 판매하거나 마케팅 목적으로 제3자에게 제공하지 않습니다. 서비스 운영에 필요한 범위에서 아래 처리자를 이용합니다.

  • Supabase — 계정·로그 데이터 저장(데이터베이스·인증)
  • Vercel — 웹 서비스 호스팅
